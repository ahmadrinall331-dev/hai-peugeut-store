## 📝 Panduan Login & Register

### **Membuat Akun Baru (Register)**
1. Klik tombol **"Login"** di header kanan
2. Klik **"Belum punya akun? Daftar sekarang"**
3. Isi form:
   - **Nama Lengkap**: Nama Anda (min 3 karakter)
   - **Email**: Email aktif Anda
   - **Password**: Minimal 6 karakter
4. Klik tombol **"Daftar"**
5. ✅ Akun berhasil dibuat!

### **Login ke Akun Existing**
1. Klik tombol **"Login"** di header
2. Isi:
   - **Email**: Email Anda
   - **Password**: Password Anda
3. Klik **"Login"**
4. ✅ Selamat datang! (nama Anda muncul di header)

### **Logout**
1. Klik nama Anda di header kanan
2. Klik **"Logout"**

## 🛍️ Cara Berbelanja (Step-by-Step)

### **Step 1: Jelajahi Produk**
- Gunakan **filter kategori** (Sepatu Lari, Sepak Bola, dll)
- Gunakan **navigasi top** (Pria, Wanita, Sepatu, Apparel)
- Gunakan **search bar** untuk cari produk spesifik
- Pilih **sorting**: Rekomendasi, Harga Terendah, Harga Tertinggi

### **Step 2: Tambah ke Keranjang**
- Lihat detail produk (harga, rating, reviews)
- Klik tombol **"+ Keranjang"**
- Angka di header akan berubah

### **Step 3: Aplikasi Diskon (Optional)**
- Klik salah satu **"EXTRA DISKON"** di hero section
- Atau pilih diskon dari **sidebar "LMS CONTROL"**
- Klik **"TERAPKAN FILTER"**

### **Step 4: Checkout**
1. Klik **ikon keranjang** di header
2. **Langkah 1 (KERANJANG)**
   - Review produk
   - Ubah jumlah dengan tombol +/-
   - Lihat total harga
   - Klik **"Lanjut →"**

3. **Langkah 2 (PENGIRIMAN)**
   - Isi **Nama Lengkap** (wajib)
   - Isi **Nomor HP** (10-13 digit, wajib)
   - Isi **Alamat Lengkap** (min 10 karakter, wajib)
   - Pilih **Metode Pengiriman**:
     - Reguler (3-5 hari) - gratis
     - Express (1-2 hari) - +Rp20.000
   - Klik **"Lanjut →"**

4. **Langkah 3 (PEMBAYARAN)**
   - Pilih **Metode Pembayaran**:
     - 💰 COD (Bayar di Tempat)
     - 🏦 Transfer Bank
     - 📱 QRIS
   - Lihat total final
   - Klik **"Pesan Sekarang"**

5. ✅ **Pesanan Berhasil!**
   - Lihat di menu **"Pesanan Saya"**

## 💾 Penyimpanan Data

### **Bagaimana Data Disimpan?**
- Semua data tersimpan di **Browser LocalStorage**
- Setiap user punya akun & data terpisah
- Data **tidak hilang** saat logout
- **Login kembali** = data kembali

### **Keamanan**
- Password di-hash sebelum disimpan
- Tidak ada data yang dikirim ke server
- 100% offline-first application

### **⚠️ Penting:**
- Data akan **dihapus** jika:
  - Clear browser cache/cookies
  - Hapus LocalStorage secara manual
  - Ganti browser/device
- **Backup data Anda** jika penting!

## 🐛 Troubleshooting

### **Akses Offline Tidak Bisa**
- Pastikan sudah buka online setidak 1x
- Browser harus support LocalStorage
- Clear cache browser & coba lagi

### **Data Hilang**
- Sudah clear browser cache?
- Sudah ganti browser/device?
- Cek apakah localStorage terusir

### **Login Tidak Bisa**
- Pastikan email & password benar
- Password case-sensitive
- Minimal password 6 karakter

### **Checkout Error**
- Pastikan sudah login
- Keranjang tidak boleh kosong
- Form pengiriman harus lengkap
